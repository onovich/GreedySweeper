# Use a CSS-native Lunar Console visual system

Greedy Sweeper will reconstruct the approved **CSS Lunar Console / 月面软仪表** direction with semantic DOM, controlled CSS primitives, and a small amount of inline SVG, using the central board and restrained amber Reward Circuit as its identifying visual system. This rejects sprite- or texture-led reproduction and the earlier photorealistic industrial hardware and vacuum-tube directions: the reduced material richness is a deliberate trade for responsive reflow, accessible states, deterministic visual fixtures, and maintainable component ownership.
